# 📖 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** on an HR dataset using Python. It focuses on cleaning, analyzing, and visualizing employee data to uncover meaningful insights that support HR decision-making.

---

# 🎯 Project Objectives

✅ Import and preprocess HR data

✅ Clean missing and inconsistent values

✅ Perform Exploratory Data Analysis (EDA)

✅ Visualize employee trends

✅ Generate actionable business insights
